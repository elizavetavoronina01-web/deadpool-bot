# 🚀 КАК ЗАПУСТИТЬ ДЕДПУЛ-БОТА

## ШАГ 1 — Подготовь файлы

В папке deadpool_bot должны быть:
- bot.py
- requirements.txt
- Procfile
- miniapp/
  - index.html
  - bg.jpg (кот с очками)
  - deadpool.jpg (дедпул на машине)
  - videos/
    - v1.mp4 ... v6.mp4

## ШАГ 2 — GitHub

1. Зайди на github.com → зарегистрируйся если нет
2. Нажми "New repository" → назови "deadpool-bot"
3. Загрузи ВСЕ файлы из папки deadpool_bot

## ШАГ 3 — Railway

1. Зайди на railway.app
2. Войди через GitHub
3. Нажми "New Project" → "Deploy from GitHub repo"
4. Выбери свой репозиторий deadpool-bot
5. Railway сам установит библиотеки и запустит бота!

## ШАГ 4 — Получи ссылку на Mini App

1. В Railway найди раздел "Settings" → "Domains"
2. Нажми "Generate Domain" — получишь ссылку типа:
   https://deadpool-bot-production.up.railway.app

3. Открой bot.py и замени строку:
   WEBAPP_URL = "https://YOUR_RAILWAY_URL/miniapp"
   На свою ссылку:
   WEBAPP_URL = "https://deadpool-bot-production.up.railway.app/miniapp"

4. Также замени CHANNEL_ID на свой канал:
   CHANNEL_ID = "@твой_канал"

5. Загрузи обновлённый bot.py на GitHub — Railway сам перезапустится

## ШАГ 5 — Настрой Mini App в BotFather

1. Открой @BotFather в Telegram
2. Напиши /mybots → выбери своего бота
3. Bot Settings → Menu Button → Configure menu button
4. Вставь ссылку: https://deadpool-bot-production.up.railway.app/miniapp
5. Напиши название кнопки: 🎰 Открыть

## ГОТОВО! 🎉

Твой бот работает! Проверь:
- Напиши /start боту
- Нажми кнопку Mini App
- Попробуй открыть кейс
